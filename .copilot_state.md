# Copilot State Tracker

Original roadmap
- Repulsive minimap markers with accurate anchors and no grouped indicators.
- Cross-instance player-presence consistency, including the red Target player buttons after movement.
- Paged Settings with a Debug page and unlimited turn times.
- Non-exploration arena PvP moving toward the same map-driven backend as exploration combat.
- Arena preview flow: 30s avatar/profile review, then a one-pass 2 minute rewrite window, then battle.
- Iterative validation with incremental commits and explicit user testing.
